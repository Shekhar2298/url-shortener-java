# URL Shortener in Java

A lightweight URL Shortener application built with:
- Java `com.sun.net.httpserver.HttpServer`
- H2 Database with JDBC
- HTML, CSS, JavaScript (no frameworks)
- SLF4J for logging
- JUnit 5 & Mockito for testing
